# Common Components

이 폴더는 여러 화면에서 반복해서 쓰는 가장 작은 UI 단위를 관리합니다. 특정 페이지의 업무 흐름보다 버튼, 카드, 입력 필드, 섹션처럼 조합 가능한 기본 요소를 우선합니다.

## 파일 안내

| 파일 | 설명 |
| --- | --- |
| `Button.tsx` | 기본 버튼과 아이콘 버튼을 제공합니다. variant, size, disabled 같은 일반 버튼 속성을 함께 다룹니다. |
| `Card.tsx` | 정보 묶음을 담는 카드 컨테이너입니다. 헤더, 본문, 하단 영역 조합에 맞춰 사용합니다. |
| `FormField.tsx` | label, 설명, 오류 문구, 입력 요소를 한 단위로 묶는 폼 필드 컴포넌트입니다. |
| `Section.tsx` | 제목과 설명, 본문 영역을 가진 화면 섹션 래퍼입니다. |
| `index.ts` | common 컴포넌트를 한 곳에서 export합니다. |

## 사용 기준

- 여러 페이지에서 같은 모양으로 반복되는 UI라면 이 폴더에 둡니다.
- 특정 기능의 상태 계산, 저장, API 처리 흐름은 넣지 않습니다.
- 화면 고유 조합이 필요하면 `features/`, `components/catalog/`, `components/showcase/` 쪽에서 감싸서 사용합니다.
- 새 공통 컴포넌트를 추가하면 `index.ts`에 export를 함께 등록합니다.
